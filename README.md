# Bilibili Transcript Viewer

一个独立项目，用来在 B 站网页视频播放时，把字幕单独拉出来显示，方便给孩子做英语启蒙、跟读和点读。

它不是桌面应用，也不是普通网页站点，而是一个 Tampermonkey 用户脚本。这样做的原因很直接：脚本运行在 B 站视频页面内部，才能稳定读取页面自带字幕，并跟当前视频播放进度同步，不会被浏览器跨域限制拦住。

## 项目内容

- `src/bilibili-transcript-viewer.user.js`：Tampermonkey 用户脚本源码
- `package.json`：本地校验和分发脚本

## 功能

- 自动读取 B 站视频已有的公开字幕
- 在页面右侧显示独立台词面板
- 顶部放大显示当前一句
- 全文列表随播放进度自动高亮
- 点击任意一句可以跳转到对应时间
- 页面内切换到另一个视频时自动重载
- 支持收起与展开
- 默认优先选英文轨，其次中文轨，更适合英语启蒙
- 支持按语音片段自动采集代表画面，并导出为 HTML 图文草稿

## 画面采集

当视频没有公开字幕时，也可以用脚本把画面采集出来，后续再补英文句子和中文释义，整理成亲子复习用的"绘本草稿"。

### 采集模式

脚本有两种采集模式，自动选择：

1. **字幕模式**（优先）：如果视频有字幕，脚本按每句字幕自动截取一张代表画面
2. **语音模式**：如果没有字幕，脚本根据实时音频强弱自动截取代表画面

### 采集策略

为了最大化关键画面捕获率，脚本采用多层采样：

- **前 3 帧**：启动采集时自动抓取视频最开头的 3 帧（可能包含封面、标题等）
- **第一段语音前**：检测到第一个声音时再额外采样 3 帧（捕捉表情变化、镜头切换等）
- **正常语音段**：每段语音结束后选择最具代表性的一帧（优先选择语音峰值处或尾部）

### 使用方式

1. 在 B 站视频页播放视频
2. 打开右侧面板，找到"绘本采集"区块
3. 点击"开始采集"，脚本会自动根据字幕或音频截取画面
4. 可随时点击"手动截取"补充当前画面
5. 播放完成或需要时点击"停止"
6. 点击"导出文档"下载包含所有截图的 HTML 或 PPTX 文件

### 导出格式

- **HTML**：内嵌 PNG 图片的单页文件，每张图片为独立卡片，便于浏览和分享
- **PPTX**：PowerPoint 演示文稿，每一张截图为一页，便于进一步编辑和演示

### 采集提示

- 脚本会采集多帧（包括重复和候选帧），建议导出后人工审视和删减
- 背景音乐较强时可能会采集更多帧，可在导出前通过 UI 删除不需要的帧
- 如果希望保留所有帧作为素材库，也可以全部导出再逐帧筛选

## 安装

1. 浏览器安装 Tampermonkey。
2. 打开 `src/bilibili-transcript-viewer.user.js`。
3. 在 Tampermonkey 里新建脚本，把默认内容替换成这个文件内容并保存。
4. 打开任意带字幕的 B 站视频页并刷新。

## 已知限制

- 只能读取视频本身已经提供的字幕（用于字幕模式）
- 语音采集依赖视频音频质量；背景音乐或环境音较强时可能影响精度
- 如果 B 站改了页面数据结构或接口，脚本需要同步调整
